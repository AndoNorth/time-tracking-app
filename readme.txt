Project Name:
  time tracking app

Description:
  an app where users can track tasks and log how much time is taken for each.

function features:
  - create containers which store:
    - name
    - time (date, hours, minutes)
    - tags (work, play, etc...)
  - store containers of data in database using JSON objects
    - save/load list states in a database
  - drag and drop containers within 3 lists
    - doing, todo, done
